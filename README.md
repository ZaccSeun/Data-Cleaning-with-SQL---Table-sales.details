# Data Cleaning with SQL

## Introduction

This project is about cleaning dirty tables (datasets) using SQL

## Project Overview
Six (6) datasets were cleaned using SQL. Key issues with the data include;
- Null/empty cells
- Duplicate records
- Presence of extra whitespaces
- Wrong date format
- Slowly Changing Dimension
- Wrong datatype
- Presence of negative values
- Presence of single letters instead of words

## Conclusion
The tables (datasets) were cleaned and the final outputs stored in new tables under schema 'final'.
